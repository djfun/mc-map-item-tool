Get all maps into the world with one command (Minecraft 1.8 and newer)

This (still experimental) feature allows you to get the map items from the main tool into your minecraft world already placed on a wall in the correct order.

To use it, simply enter the data you used to create the map item files. Open your minecraft world, place a command block (/give @p minecraft:command_block) 2 blocks in front of where the bottom left corner of the item frame wall should be. Copy the command from the output text field and paste it into the command block. Place a button onto the command block, click it and watch the block do its magic.

If that explanation was to complicated, you can just make a new creative test world and try it out.

Please note, that if you do not understand what the command does, you should probably make a backup of your world if there are important things near the location of the command block that runs this command.
